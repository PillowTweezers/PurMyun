<!-- GETTING STARTED -->

## Getting Started

### Prerequisites

* Python3.8+
* pip3

### Installation

1. Clone the repo
   ```sh
   $ git clone https://github.com/PillowTweezers/PurMyun.git
   ```
2. Install pip packages
   ```sh
   $ pip install -r requirements.txt
   ```
